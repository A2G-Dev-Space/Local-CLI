## 활용 예시

### 프로젝트 초기화

```
> Express + TypeScript로 REST API 서버를 만들어줘
> React + Vite 프로젝트를 생성해줘
> Next.js 프로젝트 구조를 잡아줘
```

### 폴더 구조 설계

```
> 클린 아키텍처 기반으로 폴더 구조를 만들어줘
> src 폴더를 모듈별로 구성해줘
> 테스트 폴더 구조도 함께 만들어줘
```

### 기본 설정 파일 생성

```
> TypeScript 설정 파일을 만들어줘
> ESLint와 Prettier 설정을 추가해줘
> Docker 및 docker-compose 파일을 만들어줘
```

### 기본 코드 작성

```
> 데이터베이스 연결 모듈을 만들어줘
> 기본 에러 핸들러를 구현해줘
> 로깅 미들웨어를 추가해줘
```

### 환경 설정

```
> .env.example 파일을 만들어줘
> 환경별 설정 파일을 구성해줘
> CI/CD 파이프라인 설정을 만들어줘
```

## 팁

- 사용할 기술 스택을 명확히 지정하세요
- 참고할 프로젝트가 있다면 구조를 먼저 분석 요청하세요
- 단계별로 진행하면서 확인하는 것을 권장합니다
