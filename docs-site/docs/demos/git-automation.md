## 활용 예시

### 커밋 자동화

```
> 변경사항 커밋해줘
> 의미있는 커밋 메시지로 커밋해줘
```

### 브랜치 관리

```
> feature/login 브랜치 만들어줘
> main 브랜치에서 최신 변경사항 가져와줘
> 이 브랜치를 main에 머지해줘
```

### PR 생성

```
> PR 만들어줘
> 변경사항 요약해서 PR 생성해줘
```

### 히스토리 분석

```
> 최근 커밋 내역 보여줘
> 이 파일의 변경 이력 확인해줘
> 누가 이 코드를 마지막으로 수정했어?
```

### 충돌 해결

```
> 머지 충돌 해결해줘
> 충돌난 파일들 확인하고 수정해줘
```

## 팁

- Nexus Coder는 변경사항을 분석하여 적절한 커밋 메시지를 자동 생성합니다
- PR 생성 시 변경 내용을 요약하여 설명을 작성합니다
- 복잡한 머지 충돌도 컨텍스트를 파악하여 해결할 수 있습니다
