## 활용 예시

### 새 React 프로젝트 시작

```
> React로 할일 관리 앱 만들어줘
> Vite로 React 프로젝트 생성하고 기본 구조 잡아줘
```

### 컴포넌트 개발

```
> 네비게이션 바 컴포넌트 만들어줘
> 카드 리스트 컴포넌트 추가해줘
> 모달 컴포넌트 구현해줘
```

### 스타일링 및 기능

```
> Tailwind CSS로 스타일링 해줘
> 다크모드 토글 기능 추가해줘
> API 연동해서 데이터 불러와줘
```

## 팁

- Browser Automation 기능으로 실시간 UI 확인이 가능합니다
- 컴포넌트 단위로 요청하면 더 정확한 결과를 얻을 수 있습니다
- 에러 발생 시 콘솔 로그를 함께 공유하면 빠른 디버깅이 가능합니다
