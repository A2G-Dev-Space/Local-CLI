## 활용 예시

### Word 문서 자동화

```
> 프로젝트 기술 문서 작성해줘
> 코드 분석 결과를 Word 문서로 정리해줘
```

### Excel 데이터 처리

```
> 로그 파일을 분석해서 Excel로 정리해줘
> API 응답 데이터를 Excel 표로 만들어줘
```

### PowerPoint 프레젠테이션

```
> 프로젝트 발표 자료 만들어줘
> 아키텍처 다이어그램이 포함된 슬라이드 생성해줘
```

### 통합 워크플로우

```
> 코드 리뷰 결과를 Word로 정리하고 Excel에 이슈 목록 만들어줘
> 분석 결과를 PPT 슬라이드로 요약해줘
```

## 팁

- Office 앱이 설치되어 있어야 합니다 (Windows 환경)
- 문서 작성 후 스크린샷으로 결과를 확인할 수 있습니다
- 복잡한 서식이 필요한 경우 단계별로 요청하세요
