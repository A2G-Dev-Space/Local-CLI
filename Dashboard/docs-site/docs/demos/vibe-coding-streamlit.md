## 활용 예시

### 새 Streamlit 앱 생성

```
> Streamlit으로 데이터 시각화 대시보드 만들어줘
> CSV 파일 업로드하고 차트로 보여주는 앱 만들어줘
```

### UI 컴포넌트 추가

```
> 사이드바에 필터 옵션 추가해줘
> 데이터 테이블과 차트를 나란히 배치해줘
```

### 기능 확장

```
> 차트 타입을 선택할 수 있게 해줘
> 데이터 다운로드 기능 추가해줘
```

## 팁

- Streamlit은 Python 기반이므로 데이터 처리와 시각화에 강점이 있습니다
- `st.` 컴포넌트들을 자유롭게 조합하여 인터랙티브한 앱을 만들 수 있습니다
- 실시간으로 변경사항을 확인하며 개발할 수 있습니다
